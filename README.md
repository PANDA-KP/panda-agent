# 🐼 Panda Agent

开源智能助手 | 记忆持久 | 人格自定义 | 数据本地化

## 功能

- 💬 智能对话
- 🧠 持久记忆（对话记录本地保存）
- 🎭 人格自定义（修改系统提示词）
- 📥 记忆导出（你的数据你做主）
- 🔒 数据本地化（不上传任何第三方）

## 快速开始

### 1. 安装Python

前往 python.org 下载安装 Python 3.10 以上版本。
安装时务必勾选 Add Python to PATH。

### 2. 下载项目

点击页面上方绿色 Code 按钮 → Download ZIP，解压到电脑任意位置。

### 3. 安装依赖

打开命令提示符（Windows按Win+R，输入cmd回车），进入项目文件夹：

pip install -r requirements.txt

### 4. 启动

streamlit run app.py

浏览器会自动打开，开始使用。

### 5. 配置API

首次使用需要在左侧填写API信息：

- API地址：https://api.siliconflow.cn/v1
- API密钥：在 siliconflow.cn 注册获取（免费）
- 模型名称：deepseek-ai/DeepSeek-V3
