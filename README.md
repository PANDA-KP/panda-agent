
# 🐼 Panda Agent

**开源智能助手 | 三层记忆 | 代码执行 | 联网搜索 | 文件分析**

Panda Agent 是一个基于小米MiMo大模型的开源智能助手，所有数据100%本地化，你的数据只属于你。

---

## ✨ 功能一览

| 功能 | 说明 | 状态 |
|------|------|------|
| 🗣️ 智能对话 | 基于小米MiMo大模型，支持中文对话 | ✅ |
| 🧠 三层记忆 | 工作记忆 + 情景记忆 + 人格记忆 | ✅ |
| 🔍 联网搜索 | 基于RSS新闻源，获取实时资讯 | ✅ |
| ⚙️ 代码执行 | 让Panda写代码并直接运行 | ✅ |
| 📁 文件分析 | 上传文件（txt/pdf/py/json等）直接分析 | ✅ |
| 🔧 内置工具 | 计算器、文本统计、历史搜索 | ✅ |
| 🎭 人格自定义 | 自定义Agent的性格和行为规则 | ✅ |
| 📥 数据导出 | 一键导出所有对话和记忆 | ✅ |
| 🔒 数据本地化 | 所有数据存储在本地，不上传任何第三方 | ✅ |
| 🛡️ 安全红线 | 不收集财务信息，不建立情感操控 | ✅ |

---

## 🚀 快速开始

### 1. 安装 Python

前往 [python.org](https://python.org) 下载 Python 3.10 以上版本。

安装时务必勾选 **Add Python to PATH**。

### 2. 下载项目

点击页面上方绿色 **Code** 按钮 → **Download ZIP**，解压到电脑任意位置。

### 3. 安装依赖

打开命令提示符（Windows按 Win+R，输入 cmd 回车），进入项目文件夹：

```bash
cd 你解压的路径\panda-agent-main
pip install -r requirements.txt

### 4. 获取API密钥

### 5. 启动
