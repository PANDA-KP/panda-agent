
# 🐼 Panda Agent

**开源智能助手 | 三层记忆 | 代码执行 | 联网搜索 | 文件分析**

Panda Agent 是一个基于小米MiMo大模型的开源智能助手，所有数据100%本地化，你的数据只属于你。

---

## ✨ 功能一览

| 功能 | 说明 | 状态 |
|------|------|------|
| 🗣️ 智能对话 | 基于小米MiMo大模型，支持中文对话 | ✅ |
| 🧠 三层记忆 | 工作记忆 + 情景记忆 + 人格记忆 | ✅ |
| 🔍 联网搜索 | 基于RSS新闻源，获取实时资讯 | ✅ |
| ⚙️ 代码执行 | 让Panda写代码并直接运行 | ✅ |
| 📁 文件分析 | 上传文件（txt/pdf/py/json等）直接分析 | ✅ |
| 🔧 内置工具 | 计算器、文本统计、历史搜索 | ✅ |
| 🎭 人格自定义 | 自定义Agent的性格和行为规则 | ✅ |
| 📥 数据导出 | 一键导出所有对话和记忆 | ✅ |
| 🔒 数据本地化 | 所有数据存储在本地，不上传任何第三方 | ✅ |
| 🛡️ 安全红线 | 不收集财务信息，不建立情感操控 | ✅ |

---

## 🚀 快速开始

### 1. 安装 Python

前往 [python.org](https://python.org) 下载 Python 3.10 以上版本。

安装时务必勾选 **Add Python to PATH**。

### 2. 下载项目

点击页面上方绿色 **Code** 按钮 → **Download ZIP**，解压到电脑任意位置。

### 3. 安装依赖

打开命令提示符（Windows按 Win+R，输入 cmd 回车），进入项目文件夹：

```bash
cd 你解压的路径\panda-agent-main
pip install -r requirements.txt

4. 获取API密钥

5. 启动

bash
streamlit run app.py

浏览器会自动打开，在左侧「模型设置」中填写：


API地址：http://
API密钥： 你的密钥
模型名称：选择大模型

点「保存设置」，即可开始对话。


📖 使用指南

基础对话

直接在底部输入框输入内容，按回车发送。


联网搜索

搜索：小米最新新闻
搜索科技动态
查一下人工智能

代码执行

让Panda写代码：
帮我写一个计算斐波那契数列的程序

然后输入 运行 即可执行Panda写的代码。


或者直接指定代码：
运行代码：print("hello panda")

文件分析

1.左侧点击「文件分析」
2.上传文件（支持 txt/pdf/py/json/csv/html/yaml 等）
3.点击「加载文件」
4.直接问问题，例如："帮我总结这个文件的要点"

内置工具
计算 123*456+789        → 数学计算
统计：你的文本内容       → 文本统计
搜索记忆：关键词         → 搜索历史对话

记忆系统

Panda有三层记忆：


L1 工作记忆： 当前对话的上下文
L2 情景记忆： 所有历史对话，可搜索
L3 人格记忆： 自动提取并记住关于你的信息（名字、职业、兴趣等）

所有记忆存储在本地 panda_data/ 文件夹中，可随时导出。


人格自定义

在左侧「人格设置」中修改系统提示词，定义Panda的性格、规则和行为方式。

🔧 技术架构

用户输入
   ↓
[工具检测层] ← 识别搜索/计算/代码执行等指令
   ↓
[记忆检索层] ← L1工作记忆 + L2情景记忆 + L3人格记忆
   ↓
[Prompt组装层] ← 系统提示词 + 用户画像 + 历史上下文
   ↓
[模型推理层] ← 小米MiMo API
   ↓
[输出与存储层] ← 流式输出 + 自动保存

🛡️ 安全原则

Panda Agent内置以下安全红线，硬编码不可绕过：


1.永远不向用户索要财务信息
2.永远不假装有情感需求
3.永远不建立情感操控关系
4.用户的隐私数据绝不外传
5.代码执行有安全限制，禁止危险操作
6.代码完全开源，欢迎审计
