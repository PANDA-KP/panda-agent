# 🐼 Panda Agent

**一个零基础小白也能用的AI助手。**

三层记忆 · 百度搜索 · 代码执行 · 图片分析 · 自动摘要 · 语义记忆 · 智能工具调用

## 快速开始

### Windows用户

1. 点击绿色 **Code** 按钮 → **Download ZIP**
2. 解压到桌面
3. 双击 `install.bat` 安装
4. 双击 `start.bat` 启动
5. 浏览器自动打开，粘贴API密钥，开始聊天

### 获取API密钥（免费）

1. 打开 [小米MiMo开放平台](https://platform.xiaomimimo.com)
2. 用手机号注册并登录
3. 点击左侧「API Keys」→「新建API Key」
4. 复制密钥，粘贴到Panda Agent

## 功能

| 功能 | 说明 |
|------|------|
| 💬 聊天 | 直接输入任何问题 |
| 🔍 百度搜索 | 输入 `搜索：关键词`，自动解析真实链接 |
| ⚙️ 代码执行 | 输入 `运行代码：...` 或 `运行` |
| 🧮 计算器 | 输入 `计算 3.14 * 12 * 12` |
| 📊 文本统计 | 输入 `统计：文本内容` |
| 📁 文件分析 | 左侧上传文档，直接问问题 |
| 🖼️ 图片分析 | 左侧上传图片，AI看懂它 |
| 🧠 记忆搜索 | 输入 `搜索记忆：关键词`，语义匹配 |
| 🤖 智能工具调用 | 开启后AI自动决定何时搜索/计算 |
| 🔄 自动摘要 | 聊天超过30条自动压缩旧消息 |
| 🔄 自动路由 | 模型断开时自动切换备用模型 |
| 🎭 人格切换 | 6种预设人格一键切换 |
| 🔑 多密钥管理 | 同模型多个密钥自动轮换 |
| 📥 数据导出 | 一键备份所有数据 |

## 搜索引擎

- **百度搜索** — 国内可用，搜索网页和新闻
- **RSS新闻源** — 新华网、36氪、IT之家、少数派
- 双引擎互补，自动过滤垃圾数据

## 数据安全

- 所有数据存储在你电脑的 `panda_data` 文件夹
- 不上传到任何第三方服务器
- 支持一键导出和清除

## 技术栈

- [Streamlit](https://streamlit.io) — 界面框架
- [小米MiMo](https://platform.xiaomimimo.com) — AI大模型
- [baidusearch](https://github.com/ur1kda/baidusearch) — 百度搜索
- [feedparser](https://github.com/kurtmckee/feedparser) — RSS解析

## 系统要求

- Python 3.8+
- Windows / macOS / Linux
- 网络连接
