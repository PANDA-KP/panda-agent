# 🐼 Panda Agent 更新日志

## v0.7 (2026-05-03)

### 重大更新
- **极简界面重构** — 左侧栏只保留新对话、历史对话、上传文件、聊天/设置两个页面
- **一键安装** — 新增 `install.bat`，双击即可安装所有依赖
- **一键启动** — 新增 `start.bat`，双击即可启动Panda Agent
- **新手引导优化** — 首次打开只需粘贴API密钥，一步到位
- **设置页面整合** — 人格选择、模型选择、密钥管理、高级设置全部收归一个页面
- **右侧面板可收起** — 默认隐藏，点击按钮展开
- **预置人格模板** — 6种人格一键切换（Panda、小智、老张、小美、教授、码农）
- **帮助中心** — 常见问题FAQ，展开式问答

### 移除
- 移除自定义技能管理页面（用预置功能替代）
- 移除独立记忆管理页面（保留自动记忆，清除功能移至设置）
- 精简左侧栏，移除统计面板和页面切换图标

### 优化
- 错误提示更友好，用大白话说明问题和解决方法
- 输入框提示更具体
- 示例按钮在欢迎页一直可见
- 每个输入框都有帮助提示

## v0.6 (2026-05-03)

### 新增
- **多密钥管理** — 同一模型可添加多个API密钥，自动轮换
- **自动路由** — 模型断开时自动切换同类别备用模型，可开关
- **模型注册表** — 下拉菜单选择模型，API地址自动填充
- **模型分类** — 通用对话、多模态、语音合成三大类
- **新手引导** — 首次打开分步引导获取API密钥
- **帮助中心** — 完整的使用教程和FAQ
- **示例提示按钮** — 聊天区可点击的示例，覆盖主要功能
- **智能错误提示** — API密钥无效、网络失败等错误给出具体解决步骤

### 优化
- 记忆管理支持手动添加、编辑、删除
- 人格管理支持预设模板一键应用
- 每个输入框增加placeholder和help提示

## v0.5 (2026-05-03)

### 新增
- **界面重做** — 米白色背景，左右分栏布局
- **右侧面板** — 搜索结果、代码输出、文件内容、工具输出分区显示
- **记忆管理页面** — 查看、添加、编辑、删除记忆
- **人格管理页面** — 预设模板 + 手动编辑
- **技能管理页面** — 内置技能开关 + 自定义技能添加
- **历史对话删除** — 每条对话可单独删除，二次确认
- **页面切换** — 聊天、记忆、人格、技能四个页面

### 优化
- 界面风格统一，不再重叠
- 侧边栏布局优化

## v0.4 (2026-05-03)

### 新增
- **联网搜索** — 基于RSS新闻源，国内可用（新华网、36氪、IT之家等）
- **代码执行** — Python代码安全沙箱执行
- **文件分析** — 支持txt/pdf/py/json/csv/html等格式
- **搜索记忆** — 在历史对话中搜索关键词
- **多种搜索触发** — 搜索/搜/查一下等格式均可触发

### 优化
- AI基于搜索结果做分析，不再说"无法联网"
